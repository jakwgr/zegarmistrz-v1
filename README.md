# Watch Service Management System
Simple Java OOP project for managing watches and their service history.
The system supports quartz and mechanical watches with different service rules.

## Features
- Add different types of watches (Quartz, Mechanical)
- Display watch information
- Track last service date
- Automatic service check based on time interval
- Service operation for watches

## Technologies
- Java
- OOP (Object-Oriented Programming)
- Collections
- LocalDate API

## How to run
Run the Main class in any Java IDE (e.g. IntelliJ IDEA)
